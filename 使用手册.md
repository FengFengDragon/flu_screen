## 二、准备工作

### 2.1 文件格式要求

| 文件类型 | 格式 | 示例 |
|----------|------|------|
| 蛋白质文件 | `.pdb` | `NP_4IDJ.pdb` |
| 小分子文件 | `.txt` / `.smi` / `.csv` | 每行一个 SMILES |

### 2.2 系统自带的PDB文件

系统在 `data/pdb/` 目录下自带了4个靶蛋白：

| 靶点 | 文件 | 说明 |
|------|------|------|
| NP（核蛋白） | `data/pdb/NP_4IDJ.pdb` | 高度保守，推荐首选 |
| HA（血凝素） | `data/pdb/HA_4WE8.pdb` | 病毒入侵细胞关键蛋白 |
| NA（神经氨酸酶） | `data/pdb/NA_4WE8.pdb` | 奥司他韦靶点 |
| PB2 | `data/pdb/PB2_5E7T.pdb` | 病毒复制相关 |

### 2.3 小分子 SMILES 格式

在文本文件中每行写一个 SMILES 字符串：

```
C1=CC(=CC=C1C2=CC(=O)C3=C(O2)C=C(C(=C3O)[C@H]4[C@@H]([C@H]([C@@H]([C@H](O4)CO)O)O)O)O)O
CC(=O)Oc1ccccc1C(=O)O
CC(C)Cc1ccc(cc1)C(C)C(=O)O
```

> 可以从 [PubChem](https://pubchem.ncbi.nlm.nih.gov/) 或 [ZINC](https://zinc.docking.org/) 获取 SMILES。

## 三、预测蛋白质残基信息

> 访问页面：`http://localhost:5000/ml-binding`

### 3.1 操作步骤

**第一步：上传PDB文件**

1. 打开「机器学习」页面
2. 在「预测」标签页中，点击文件上传区域
3. 选择你的 `.pdb` 文件（如 `NP_4IDJ.pdb`）

**第二步：选择算法**

从下拉菜单中选择算法：

| 算法 | 特点 | 适用场景 |
|------|------|----------|
| 随机森林 | 可解释性强，速度快 | 一般用途（推荐） |
| SVM | 适合小样本 | 数据量较少时 |
| 梯度提升 | 精度最高 | 追求精度时 |
| 启发式方法 | 无需训练，秒出结果 | 快速预览 |

**第三步：调整阈值**

- 拖动滑块设置预测阈值（默认 0.5）
- 阈值越低 → 预测出的结合残基越多
- 阈值越高 → 只保留高置信度残基

**第四步：点击「开始预测」**

### 3.2 结果解读

预测完成后会显示：

- **总残基数**：蛋白质中的氨基酸数量
- **预测结合残基数**：被判定为结合位点的残基数量
- **残基列表**：每个残基包含以下信息：
  - `residue_name`：氨基酸名称（如 ARG、TYR）
  - `residue_num`：残基序号
  - `chain_id`：链编号
  - `probability`：预测为结合位点的概率（0~1）
  - `is_binding`：是否为结合位点

### 3.3 特征分析（可选）

切换到「特征分析」标签页：

1. 上传同一个 PDB 文件
2. 点击「分析特征」
3. 查看每个残基的 32 维特征信息，包括：
   - 氨基酸类型（20维 one-hot 编码）
   - 疏水性、电荷、极性、体积
   - 溶剂可及性、B因子、深度
   - 邻居数量、邻居密度
   - 疏水邻居比例、带电邻居比例
   - 序列位置比例

## 四、小分子虚拟筛选与对接

> 访问页面：`http://localhost:5000/virtual-screening`

### 4.1 方式A：文件上传一键筛选（推荐）

**第一步：切换到「完整筛选」标签页**

**第二步：上传小分子文件**

1. 找到「📁 上传文件并直接筛选」区域
2. 选择你的 `.txt` / `.smi` / `.csv` 文件
3. 文件中每行一个 SMILES

**第三步：设置筛选参数**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| Tier1 保留数量 | 500 | 高通量筛选保留数量 |
| Tier2 保留数量 | 100 | 标准对接保留数量 |
| Tier3 保留数量 | 20 | 精细对接保留数量 |
| 靶蛋白 PDB 路径 | 空 | 可选，填PDB文件绝对路径 |

**第四步：点击「上传并筛选」**

等待处理完成，页面会显示筛选结果。

### 4.2 方式B：手动输入 SMILES

**第一步：在「预处理」标签页**

1. 先在「上传文件」区域上传文件，点击「解析文件」
2. 解析后的 SMILES 会自动填入文本框
3. 或直接在文本框中手动输入 SMILES（每行一个）

**第二步：勾选过滤器**

| 过滤器 | 说明 |
|--------|------|
| Lipinski 五规则 | 分子量≤500、LogP≤5、HBD≤5、HBA≤10 |
| ADMET 过滤 | 吸收、分布、代谢、排泄、毒性 |
| PAINS 过滤 | 排除干扰化合物 |

**第三步：点击「开始预处理」**

**第四步：切换到「完整筛选」标签页**

手动输入 SMILES，设置参数后点击「开始筛选」。

### 4.3 结果解读

筛选完成后显示：

| 指标 | 说明 |
|------|------|
| 对接分数（docking_score） | 基于描述符的综合评分，越高越好 |
| MM-GBSA 能量 | 结合自由能估算（kcal/mol），越负越好 |
| Kd 值 | 估计的解离常数（μM），越小越好 |
| 置信度 | high / medium，high 表示高置信度 |

**判断标准**：

- MM-GBSA < -45 kcal/mol → 极高潜力
- MM-GBSA < -35 kcal/mol → 高潜力
- MM-GBSA < -25 kcal/mol → 中等潜力
- MM-GBSA > -25 kcal/mol → 低潜力

## 五、单分子详细分析

> 访问页面：`http://localhost:5000/virtual-screening` → 「单分子分析」标签页

### 5.1 操作步骤

1. 在文本框中输入一个 SMILES
2. 点击对应的分析按钮：

| 按钮 | 功能 | 返回信息 |
|------|------|----------|
| 📊 描述符 | 计算分子描述符 | MW、LogP、TPSA、QED等 |
| 💊 Lipinski | Lipinski五规则检查 | 违反项列表 |
| 🧬 ADMET | ADMET性质预测 | 口服吸收、血脑屏障等 |
| ⚠️ PAINS | PAINS过滤 | 是否为干扰化合物 |
| ⚡ 结合能 | 结合自由能计算 | MM-GBSA能量、Kd值 |
| 🔗 结合模式 | 结合模式分析 | 功能基团、类药性、相互作用预测 |
| 🧊 3D构象 | 生成3D构象 | 原子坐标 |

## 六、完整工作流示例

### 场景：从PDB蛋白和SMILES小分子出发，完成全套分析

```
步骤1：上传PDB → 机器学习预测残基
        ↓
步骤2：上传小分子文件 → 虚拟筛选
        ↓
步骤3：查看Top候选分子 → 单分子详细分析
        ↓
步骤4：结合残基信息 + 对接结果 → 综合判断
```

#### 具体操作：

**Step 1：残基预测**

1. 打开 `http://localhost:5000/ml-binding`
2. 上传 `NP_4IDJ.pdb`
3. 算法选「随机森林」，阈值 0.5
4. 点击「开始预测」
5. 记录预测出的关键残基（如 ARG-150、TYR-148 等）

**Step 2：小分子筛选**

1. 打开 `http://localhost:5000/virtual-screening`
2. 切换到「完整筛选」标签页
3. 上传包含 SMILES 的 `.txt` 文件
4. 点击「上传并筛选」
5. 查看 Top 候选分子

**Step 3：单分子分析**

1. 切换到「单分子分析」标签页
2. 输入 Top 分子的 SMILES
3. 依次点击：
   - 「⚡ 结合能」→ 查看 MM-GBSA 能量
   - 「🔗 结合模式」→ 查看功能基团和相互作用
   - 「📊 描述符」→ 查看分子性质

**Step 4：综合判断**

结合残基预测结果和对接结果，判断：
- 小分子是否能与关键残基相互作用
- 结合自由能是否足够低（< -35 kcal/mol）
- 分子是否符合药物性质（Lipinski、ADMET）
